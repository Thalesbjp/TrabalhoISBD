## Como Testar o projeto

### 1 - instale o MySqlWorkBanch com SQLServer
    - https://www.mysql.com/downloads/
    - Configure a senha para 2022#estudante ou altere o arquivo .\Academia\Config.php com sua senha
    - Execute os scrips SQL da pasta Scripts no MySqlWorkbanch (cria o banco, tabelas povoa, trigguers, Views)

### 2 - Instale o servidor XAMP
    - https://www.apachefriends.org/pt_br/index.html
    - copie a Pasta academia para a Pasta C:\xampp\htdocs\
    - Inicie o shamp

### 3 - Testando a Página
    - http://localhost/Academia/index.php
